# 🔰 Number System Converter 🔰

## Description

This program allows the user to convert numbers between different numeral systems such as:

- Binary (base 2)
- Octal (base 8)
- Decimal (base 10)
- Hexadecimal (base 16)

The application provides a simple graphical interface where the user can enter a number, select the source number system and obtain the result in another system.

## Technologies

- C++
- Qt Framework
- Qt Creator

## Project Structure

- `main.cpp` — program entry point  
- `mainwindow.cpp / mainwindow.h` — main window logic  
- `mainwindow.ui` — graphical interface  
- `logic.cpp / logic.h` — number conversion logic  
- `entrypoint.cpp / entrypoint.h` — application interaction logic  
- `Converter.pro` — Qt project configuration
